# Gatito vs Perrito 🐱🐶 (Juego de Gato)

Una versión sumamente tierna y con tonos pastel del clásico juego de gato (tres en línea / Tic-Tac-Toe), donde los jugadores compiten como Gatito (🐱) o Perrito (🐶).

## 🌟 Características
- **Diseño Aesthetic & Pastel:** Interfaz tierna con animaciones suaves y un esquema de colores rosa, azul y amarillo.
- **Efectos de Sonido:** Sonidos retro y agradables generados en tiempo real con la Web Audio API (¡sin archivos de audio externos!).
- **Control de Sonido:** Botón para activar o desactivar el sonido en cualquier momento.
- **Marcador de Puntuación:** Lleva el registro en tiempo real de las victorias del Gatito, del Perrito y los empates.
- **Controles Rápidos:** Botones para reiniciar la partida actual o borrar todo el historial del marcador.
- **Diseño Responsivo:** Optimizado para jugarse cómodamente en dispositivos móviles, tablets y computadoras.

## 🚀 ¿Cómo jugar?
No necesitas instalar nada ni configurar un servidor. Simplemente:

1. Descarga el repositorio o los archivos.
2. Abre el archivo `index.html` en tu navegador web preferido (Chrome, Firefox, Safari, Edge, etc.).
3. ¡Empieza a jugar! El jugador actual se indica en la parte superior.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica del juego.
- **CSS3:** Estilos personalizados, gradientes y animaciones fluidas.
- **JavaScript (ES6):** Lógica del juego y síntesis de audio a través de la **Web Audio API**.
